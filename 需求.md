### JieGer 个人主页 V2.1 最终需求文档

#### 1. 项目概述
- **项目名称**：JieGer 个人开发者主页
- **核心定位**：HarmonyOS 独立开发者数字名片
- **设计原则**：极简优雅、克制专注、隐私优先、系统原生感
- **交付形态**：单文件 HTML（零构建依赖，双击即用，支持直接部署）

#### 2. 视觉与设计规范
| 属性 | 规范值 | 说明 |
| :--- | :--- | :--- |
| 背景色 | `#F8F9FC` | 柔和蓝灰白，降低长时间阅读疲劳 |
| 正文色 | `#2D3748` | 深灰蓝，保证 WCAG AA 级对比度 |
| 强调色 | `#3B5BDB` | 鸿蒙蓝，仅用于按钮、编号、交互态 |
| 辅助色 | `#A0AEC0` | 用于标签、次要文字、预留入口默认态 |
| 卡片背景 | `#FBFCFE` | 极浅灰白，避免纯白在浅色背景上对比过强 |
| 字体栈 | `Inter`, `Noto Sans SC`, sans-serif | 英文优先 Inter，中文回退 Noto Sans SC |
| 圆角规范 | 按钮 8px / 标签 99px / 卡片 12px | 统一视觉语言，贴合鸿蒙原生设计 |
| 阴影规范 | `rgba(59,91,219,0.04)` | 仅卡片使用，hover 时加深至 0.08 |

#### 3. 页面结构与内容规范
页面采用单页纵向布局，严格遵循以下顺序：

1.  **Hero 首屏**
    - 标题：`Hi, I'm JieGer.`（clamp 响应式字号）
    - 副标题：`独立开发者 · 用理念驱动鸿蒙原生应用`（单行显示）
    - CTA 按钮：`与我联系`（mailto 链接，强调色填充）
2.  **关于我**
    - 精炼 1-2 句话，阐述鸿蒙原生开发理念与克制态度
3.  **技术栈**
    - 居中展示 5 个标签：ArkTS / DevEco Studio / HarmonyOS SDK / TypeScript / UI/UX Design
    - 默认态：辅助色边框+文字；Hover 态：强调色边框+文字
4.  **开发理念（四宫格卡片）**
    - 01 尊重系统原生体验：不强行移植 iOS/Android 逻辑，深度融入 HarmonyOS 设计规范
    - 02 克制与专注：每个功能都有明确目的，拒绝“为了创新而创新”
    - 03 隐私优先：默认最小权限，数据本地处理，透明可控
    - 04 为长期使用而构建：代码可维护、架构可演进，不追求短期交付
5.  **预留入口（关键 SEO 内链）**
    - 文案：`探索技术笔记 →`
    - 位置：开发理念下方、联系区上方，居中显示
    - 样式：默认辅助色，hover 变强调色，下划线动画过渡，无边框/背景/图标
    - 链接：`/notes`（语义化锚点，为后续博客页面预留）
6.  **联系我**
    - 引导文案：`有合作想法或技术交流？欢迎随时联系`
    - 邮箱：`healwis@qq.com`（点击自动复制到剪贴板并弹出提示）
7.  **页脚**
    - 文案：`© 2026 JieGer.`
    - 位置：页面底部居中，上方带 1px 分隔线

#### 4. SEO 与性能硬性要求
- **语义化 HTML**：必须使用 `<header>`, `<main>`, `<article>`, `<footer>` 等语义标签嵌套
- **JSON-LD 结构化数据**：包含 Person Schema（姓名、职位、邮箱、技能、描述）
- **Meta 标签**：完整覆盖 title / description / keywords / robots / author
- **社交分享**：Open Graph + Twitter Card 双套标签，确保社交平台预览正常
- **性能指标**：除 Google Fonts CDN 外无任何外部请求，Lighthouse 性能分 ≥95，CLS=0
- **无障碍**：所有交互元素具备 `aria-label`，颜色对比度符合 WCAG AA 标准

#### 5. 响应式适配规则
| 断点 | 适配策略 |
| :--- | :--- |
| ≥1024px | 最大宽度 720px 居中，四宫格 2×2 布局 |
| 641-1023px | 容器自适应，四宫格保持 2×2，字号微调 |
| ≤640px | 四宫格转为单列，Hero 区上下间距缩减，标签换行居中 |

#### 6. 交付验收标准
- [ ] 双击 HTML 文件可在 Chrome/Safari/Edge 中完美渲染
- [ ] 移动端真机测试无横向滚动、文字溢出、触摸目标过小问题
- [ ] 浏览器开发者工具验证 JSON-LD 与 Meta 标签完整无误
- [ ] 点击邮箱可触发复制提示，预留入口 hover 动效流畅
- [ ] 取色器验证所有颜色值与本文档第 2 节完全一致
- [ ] Lighthouse 测试性能、无障碍、SEO 三项均 ≥90
